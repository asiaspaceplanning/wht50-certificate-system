วิธีแก้กรณีหน้าเว็บขึ้นว่าไม่สามารถโหลดข้อมูลระบบได้ และมีข้อความ HTML ยาว

1. อัปโหลดชุดไฟล์นี้ขึ้น GitHub ทับของเดิม หรืออย่างน้อยให้อัปโหลด app/page.tsx และ app/api/app/route.ts ทับของเดิม
2. ไปที่ Vercel > Deployments > Redeploy
3. หลัง Deploy สำเร็จ เปิด URL: https://ชื่อเว็บของคุณ.vercel.app/api/app
4. ถ้าถูกต้องต้องเห็นข้อความ JSON: {"ok":true,"message":"ช่องทางเชื่อมต่อข้อมูลพร้อมใช้งาน",...}
5. จากนั้นกลับไปหน้าเว็บหลักและเข้าสู่ระบบด้วย APP_ACCESS_CODE
